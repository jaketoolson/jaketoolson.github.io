Kabooodle.com

Direct sales SaaS I created for my wife in 2016, to help her manage her direct sales inventory, and track and ship sales. Took about 6 months to complete and launch.

  • Custom claiming system
  • Payment gateway for handling shipping charges and invoices
  • USPS shipping integration and label creation
  • SMS Notifications
  • "Like" and "Watch" items
  • Referral and rewards system
  • Custom sales analytics
  • Facebook integration
  • Direct messaging system
  • Inventory grouping (build outfits with your existing inventory)
  • LEMP (Ubuntu, Nginx, MariaDB, PHP7)
  • Laravel 5.x MVC Framework
  • Pusher - websocket, realtime push notifications
  • Webpack, ES6, Vue/x - Frontend components
  • Async queues with over 25 pipelines
  • AWS:
    • EC2 - Custom AMI instances, multizone
    • EC2 bastion - tunneling into system remote
    • S3 - File storage with IA for thumbnails
    • ELB - Public facing Load Balancer
    • LAMBDA - Image thumbnail creation, file manipulation
    • SQS - Cron task runner
    • Cloudfront - CDN for all assets
    • RDS - MariaDB, multizones
    • ElastiCache - Redis cluster, multizone
    • VPC - All services behind VPC private subnets.
  • Keen.io: Track user and sale analytics
